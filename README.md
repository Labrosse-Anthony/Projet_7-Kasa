## Kasa
- Ce travail a été réalisé dans le cadre du projet n°6 de la formation Intégrateur Web d’OpenClassrooms.

### Contexte :
- Kasa est dans le métier de la location d’appartements entre particuliers depuis près de 10 ans. L'entreprise souhaite faire une refonte totale de son site web.

## Outils et langages pour la réalisation du projet

- Le projet a été réalisé en **HTML**, **CSS** , **JavaScript** et **React**
